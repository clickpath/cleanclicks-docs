CleanClicks sends conversion events to TikTok via the Events API. Like Meta, TikTok uses a manually generated access token.

## Prerequisites

- A TikTok Ads Manager account
- A TikTok Pixel created in Events Manager
- Access to generate API tokens

## Getting Your Credentials

### Pixel Code
1. Go to TikTok Ads Manager > **Assets** > **Events** > **Web Events**
2. Select your pixel
3. The Pixel Code is displayed on the overview page (format: `CXXXXXXXXXXXXXXXXX`)

### Access Token
1. On the same pixel settings page, look for **Generate Access Token**
2. Click to generate a token
3. Copy it immediately (you may not be able to see it again)

## Connecting in CleanClicks

1. Go to **Connections** in your CleanClicks dashboard
2. Select your domain
3. Find **TikTok** in the Token Entry section
4. Paste your **Access Token**
5. Enter your **Pixel Code**
6. Click **Save**

## Configuring the Vendor

After connecting:

1. Go to **Configuration** > **Vendors** tab
2. Enable **TikTok**
3. Verify the Pixel Code and Access Token are present
4. Save

## How It Works

When a visitor converts:
1. CleanClicks captures the `ttclid` (TikTok Click ID) from the landing page URL
2. The conversion event is sent server-side to TikTok's Events API with the ttclid
3. TikTok matches the conversion to the original ad click
4. Customer data is sent as SHA-256 hashes for Advanced Matching

## Limited Data Use

For visitors subject to CCPA, CleanClicks automatically sends `limited_data_use: true` to TikTok, restricting how TikTok processes that visitor's data.

## Troubleshooting

| Issue | Solution |
|---|---|
| Token rejected | Regenerate the access token in TikTok Events Manager |
| Events not appearing | Verify the Pixel Code matches your pixel. Check that TikTok is enabled in Configuration > Vendors. |
| Pixel Code format | Must start with `C` followed by alphanumeric characters (e.g., `CXXXXXXXXXXXXXXXXX`) |
